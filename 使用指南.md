# 📊 售后数据分析后台 — 使用指南

## 一、系统简介

本系统用于**舜洁（SooJee）售后配件记录**的月度分析与多月对比，支持：

- 📤 **上传 Excel** → 自动解析日期、拆分多月份、执行 12 项分析
- 📋 **历月数据** → 查看/删除/导出各月分析报告
- 📈 **多月对比** → 趋势图表 + AI 智能分析建议
- 💾 **Excel 导出** → 单月报告下载为格式化 Excel

---

## 二、运行环境要求

### 方案 A：一键版（无需安装 Python）⭐ 推荐

直接拷贝 `售后分析系统` 文件夹到任意 Windows 电脑，双击 `启动.bat`。

> 已内置 Python + 全部依赖，解压即用。

### 方案 B：源码版（需要 Python 3.8+）

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
python app.py
```

浏览器访问 **http://localhost:5859**

## 三、快速上手

### 3.1 准备数据

Excel 文件要求：

| 项目 | 要求 |
|------|------|
| 文件格式 | `.xlsx`（不支持 .xls） |
| 文件名 | 含年月，如 `202509售后配件记录.xlsx` |
| 必需列 | 反馈日期、盖机/整机、使用月数、保修、部品费、运费、型号、配件 |
| 日期格式 | 支持 `9.1`（9月1日）、`2025-09-01`、`9-1` 等 |

### 3.2 上传分析

1. 打开系统，进入「📤 上传分析」页
2. 拖拽或点击上传 Excel 文件
3. 系统自动检测单月/多月数据：
   - **单月**：直接展示分析结果
   - **多月**：逐月拆分展示，已存在的月份弹出覆盖确认
4. 可滚动查看全部 12 项分析表格

### 3.3 历月数据管理

- 「📋 历月数据」页查看所有已保存月份
- 点击月份卡片查看详情、导出 Excel
- 可删除不需要的月份数据

### 3.4 多月对比

- 「📈 多月对比」页自动汇总所有月份的对比图表
- 包含趋势图：记录数、费用、保内占比、盖机/整机对比
- 包含 AI 智能分析结论、风险预警、改进建议

## 五、一键移植说明

### 5.1 文件夹结构

```
售后分析系统/
├── 启动.bat          ← 双击启动
├── 售后分析后台.exe   ← 主程序（已打包）
├── frontend/
│   ├── index.html
│   └── logo.png
├── backend/
│   ├── after_sales.db  ← 数据库（自动创建）
│   └── uploads/        ← 上传文件存放
└── 使用指南.md
```

### 5.2 移植到新电脑

1. 将 `售后分析系统` 整个文件夹复制到目标电脑
2. 双击 `启动.bat`
3. 浏览器打开 http://localhost:5859

### 5.3 构建 exe（开发者用）

```bash
# 安装 PyInstaller
pip install pyinstaller

# 执行构建
build.bat
```

## 六、常见问题

**Q: 端口 5859 被占用？**
修改 `launch.bat` 中的端口号。

**Q: 数据库在哪里？**
`backend/after_sales.db`，如需备份直接复制该文件。

**Q: 上传文件后浏览器显示不完整？**
按 Ctrl+F5 强制刷新。
